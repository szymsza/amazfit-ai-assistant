Amazfit Voice Assistant
=======================

Voice AI assistant for Amazfit Balance running on Zepp OS.
Ask questions by voice, get spoken answers from Claude or ChatGPT.

Architecture
------------
Monorepo with two components:

  app/    - Zepp OS mini-app (JavaScript, runs on watch + phone Side Service)
  server/ - Backend API (single endpoint: receives audio, returns audio)

How it works:

  1. Watch: user presses button, records voice (OPUS via Recorder API)
  2. Watch -> Phone: audio sent over BLE to Side Service (runs inside Zepp companion app)
  3. Phone -> Server: Side Service makes single fetch() POST with audio bytes
  4. Server pipeline: STT (Groq Whisper) -> LLM (Claude or ChatGPT) -> TTS (OpenAI TTS)
  5. Server -> Phone: returns audio response (MP3/OPUS)
  6. Phone -> Watch: audio transferred back over BLE
  7. Watch: plays response via Player API

No separate Android app needed - Side Service runs inside the Zepp companion app.

Key APIs used (watch side):
  - Recorder API (OPUS) for voice capture
  - Player API (MP3/OPUS) for audio playback
  - Messaging/TransferFile API for watch <-> phone BLE communication
  - Fetch API (Side Service) for HTTP requests to our server
